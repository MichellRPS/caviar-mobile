Documentação da API

Retorna todos os itens cadastrados

  GET /read.php/${tabela}

Substituir {tabela} pelo nome da tabela.

Com excessão da tabela enderecos_clientes:

  POST /read.php/enderecos_clientes?id_cliente=${parametros}
